# Filmo Landing Demo

Mini landing demo created to learn GitHub, commits, pull requests and GitHub Pages as a Product Designer.

## Goal

Practice a basic product workflow:

Figma → AI-assisted code → Commit → Pull Request → Deploy

## What I want to learn

- How repositories work
- How to create meaningful commits
- How to work with branches
- How to open and review Pull Requests
- How to publish a simple landing with GitHub Pages
